# RuyiPage Dev Skill

这个 skill 用于在 Codex 中开发、调试和重构 RuyiPage 的 Python 自动化脚本，重点覆盖 Firefox 路线、BiDi 能力、接管已有浏览器、元素交互、上下文隔离、网络拦截和高风控场景。

## 内容组成

- `SKILL.md`
  - 触发描述和工作流说明
- `agents/openai.yaml`
  - Skill 的界面元数据
- `references/project-readme.md`
  - 项目 README 的本地副本
- `references/index.md`
  - 本地参考资料导航
- `references/docs/`
  - 从官方文档站抓取并清理后的 RuyiPage 文档

## 引用来源

- 文档站引用：<https://0xshoulderlab.site/automation>
- 项目仓库引用：<https://github.com/LoseNine/ruyipage>

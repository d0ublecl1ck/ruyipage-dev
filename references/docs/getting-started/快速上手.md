#  RuyiPage 快速上手
这一页专门回答新用户的两个问题：如何最快跑通，以及如何把最短脚本扩展到真实业务页面。
## 推荐顺序
  * 先跑 examples/01_quickstart.py，确认环境和 Firefox 启动正常。
  * 再跑 examples/01_basic_navigation.py，理解页面对象和基础导航。
  * 随后看 quickstart_bing_search.py，把输入、回车、列表提取串起来。
  * 最后看 quickstart_cloudfare.py，理解高风控站点下的真实实战路径。


## 完整示例源码（01_quickstart.py）

自动换行 复制代码
```
from ruyipage import launch

def quickstart_demo():
    # launch() 是新手友好入口，自动创建 FirefoxOptions 并启动 Firefox
    # headless=False 表示有界面模式，调试时推荐
    page = launch(headless=False)

    try:
        # page.get(url) 导航到指定页面，默认等待 readyState == 'complete'
        page.get("https://example.com")

        # page.wait(seconds) 等待指定秒数
        page.wait(1)

        # page.title 返回当前页面 <title> 文本
        print(f"标题: {page.title}")

        # page.ele(locator) 查找单个元素
        # 'tag:h1' 表示按标签名定位，返回 FirefoxElement
        h1 = page.ele("tag:h1")

        # ele.text 返回元素的 textContent 文本
        print(f"H1文本: {h1.text}")
    finally:
        # page.quit() 关闭浏览器并清理资源
        page.quit()

if __name__ == "__main__":
    quickstart_demo()
```

## 核心 API
API | 说明  
---|---  
launch(headless, port, browser_path, user_dir, window_size) | 快速启动 FirefoxPage，新手首选入口，内部自动创建 FirefoxOptions  
attach(address) | 连接已启动的 Firefox 调试端口，不重新启动浏览器  
page.get(url, wait='complete') | 导航到 URL，wait 可选 'complete'/'interactive'/'none'  
page.title | 当前页面标题（str）  
page.url | 当前页面 URL（str）  
page.html | 当前页面完整 HTML（str）  
page.wait(seconds) | 等待指定秒数  
page.ele(locator) | 查找单个元素，返回 FirefoxElement；未找到返回 NoneElement  
page.eles(locator) | 查找所有匹配元素，返回列表  
page.quit() | 关闭浏览器并清理单例缓存

# RuyiPage 示例注释版
# 标题: Bing 搜索并抓取前三页结果
# 说明: 最接近真实业务脚本的入门示例，展示如何从搜索框输入到结果列表提取。

#   1. 打开搜索引擎主页并定位输入框

#   5. 跨页面翻页抓取结果

# -*- coding: utf-8 -*-
"""快速开始：Bing 搜索前 3 页。"""
from ruyipage import FirefoxOptions, FirefoxPage, Keys


opts = FirefoxOptions()
# 如果 Firefox 不在默认安装目录，可以取消注释并指定路径。
# opts.set_browser_path(r"D:\Firefox\firefox.exe")

# 如果你想复用登录状态、Cookie、扩展，可以取消注释并指定 userdir。
# opts.set_user_dir(r"D:\ruyipage_userdir")

page = FirefoxPage(opts)

try:
    page.get("https://cn.bing.com/")
    page.ele("#sb_form_q").input("小肩膀教育")
    page.actions.press(Keys.ENTER).perform()
    page.wait(3)

    for page_no in range(1, 4):
        print("=" * 80)
        print(f"第 {page_no} 页")
        print("=" * 80)

        items = page.eles("css:#b_results > li.b_algo")

        for i, item in enumerate(items, 1):
            title_ele = item.ele("css:h2 a")
            if not title_ele:
                continue

            title = " ".join((title_ele.text or "").split())
            url = title_ele.attr("href") or ""

            desc_ele = item.ele("css:.b_caption p")
            content = desc_ele.text if desc_ele else item.text
            content = " ".join((content or "").split())

            print(f"{i}. {title}")
            print(f"   URL: {url}")
            print(f"   内容: {content}")

        if page_no < 3:
            next_btn = page.ele("css:a.sb_pagN")
            if not next_btn:
                break
            next_btn.click_self()
            page.wait(2)
finally:
    page.quit()
